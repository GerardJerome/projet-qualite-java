# Projet d'introduction à JUnit et Sonar

## Objectifs
- Découvrir et écrire des tests unitaires JUnit
- Analyser la qualité du code avec SonarLint ou SonarQube
- Améliorer le code existant et corriger les problèmes détectés

## Étapes
1. Cloner ou ouvrir le projet
2. Lancer les tests existants
3. Compléter la classe BankAccount
4. Ajouter des tests dans BankAccountTest
5. Lancer SonarLint ou SonarQube
6. Corriger les problèmes et augmenter la couverture

## Commandes utiles
```bash
mvn test
mvn clean verify
